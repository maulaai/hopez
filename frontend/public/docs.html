Documentation

Everything you need to integrate HOPEZ.AI in under five minutes.

1. Create an account

Sign up at /signup. You'll receive 100 free credits and the ability to issue one API key — enough to evaluate the full platform.

2. Generate an API key

Open the Console, choose API Keys, and click Create key. The full secret is shown only once; store it securely. Keys are prefixed hpz_aidfz_.

3. Make your first request

The HOPEZ.AI API is plain HTTPS + JSON. Point any HTTP client at https://api.hopez.ai/v1 and use your HOPEZ key.

Node.js

const res = await fetch("https://api.hopez.ai/v1/chat/completions", {
  method: "POST",
  headers: {
    "Authorization": `Bearer ${process.env.HOPEZ_API_KEY}`,
    "Content-Type": "application/json"
  },
  body: JSON.stringify({
    model: "hopez-mini",
    messages: [{ role: "user", content: "Say hello in French." }]
  })
});
const data = await res.json();

Python

import os, requests

resp = requests.post(
    "https://api.hopez.ai/v1/chat/completions",
    headers={
        "Authorization": f"Bearer {os.environ['HOPEZ_API_KEY']}",
        "Content-Type": "application/json",
    },
    json={
        "model": "hopez-mini",
        "messages": [{"role": "user", "content": "Hello!"}],
    },
)
data = resp.json()

cURL

curl https://api.hopez.ai/v1/chat/completions \
  -H "Authorization: Bearer $HOPEZ_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "hopez-mini",
    "messages": [{"role":"user","content":"Hello"}]
  }'

Supported endpoints

  • POST /v1/chat/completions
  • POST /v1/embeddings
  • GET  /v1/models

Authentication

Include your key as a bearer token: Authorization: Bearer hpz_aidfz_…. Each request deducts credits from your account balance based on the model's published rate.

Rate limits

Free tier: 60 requests/minute per key. Paid tiers scale to 600 RPM by default; enterprise users may request a higher limit.

Errors

HOPEZ.AI returns standard HTTP status codes:

  • 401 — invalid or revoked API key
  • 402 — insufficient credits
  • 429 — rate limit exceeded
  • 5xx — service temporarily unavailable
